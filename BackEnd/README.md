# AllEn
这是后端仓仓库，使用 C# 和 .NET 8 构建。

## 项目结构
在4-14目录下
- `Controllers/` - 存放 API 控制器，处理 HTTP 请求和响应。
- `Models/` - 存放项目的数据模型类。
- `Services/` - 存放业务逻辑服务类。

## 安装和运行

### 前提条件
- .NET 8 SDK已安装

### 克隆仓库
如果你已经配置了SSH
```bash
git clone git@github.com:Notyourbing/AllEnBackEnd.git
``` 

### 运行
在根目录下点击AllEnBackend.sln，在Visual Studio 2022中运行。
